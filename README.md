# Game2D

Game2D is an open source 2D game engine for Java.<br>
Our focus currently lies on getting this project to a usable state.<br>
We would be happy about contributions to this project.

## Getting started

**Click [here](/md/GettingStarted.md), to get to our starter guide.**

## Development progress

### In progress

- **Updated collision system**
- **Documentation**
- **Chunk system**
- **Migrate render system to functionality provided by the chunk system**

### Planned

- **Interface improvements**
- **Fixing the demos**
- **OpenGL based renderer**
- **Sound engine**

## Download

To currently use Game2D, you either have to compile the project by yourself, or you have to clone this repository and base your work on to it.
<p>In the future, Game2D will be available as a maven package.</p>
